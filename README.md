# AI Email Triage and Task Manager

An AI-powered workflow automation system built using n8n to classify emails, extract tasks, prioritize actions, and manage productivity automatically.

## Features
- Email classification and prioritization
- AI-based task extraction
- Automated task creation in Todoist
- Google Sheets logging
- Email summarization
- Workflow automation using n8n

## Workflow
1. Fetch emails
2. Analyze using LLM
3. Extract action items
4. Categorize by priority
5. Store in Google Sheets
6. Create tasks in Todoist

## Tech Stack
- n8n
- Google Sheets API
- Todoist API
- Gemini / OpenAI API
- JavaScript
- Workflow Automation

## Use Cases
- Productivity optimization
- Task automation
- Smart email management
- AI-powered workflow systems
